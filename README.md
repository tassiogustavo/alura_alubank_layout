# Curso Alura - Revisão de Widgets e Estilização em Flutter

![alubank](https://github.com/tassiogustavo/alura_alubank_layout/assets/70405308/1d7126cd-365c-4883-9af5-c3dd7c91553f)

Este projeto foi desenvolvido como parte do curso da Alura "Flutter: estilizando e reproduzindo layouts", que faz parte da formação de Flutter. O objetivo deste projeto é revisar conceitos essenciais de estrutura de widgets, estilização, organização de código, alteração de tema do projeto e adição de fontes personalizadas em aplicativos Flutter.

## Descrição do Projeto

O projeto consiste em um aplicativo de exemplo que demonstra como revisar e aprimorar a estrutura de widgets de um aplicativo Flutter, bem como melhorar a estilização e organização do código-fonte. Além disso, o projeto aborda a alteração do tema do aplicativo para personalizá-lo e a adição de fontes diferentes para uma aparência única.

## Conteúdo do Curso

O curso da Alura abrange os seguintes tópicos:

- Revisão de Widgets de Estrutura
- Revisão de Widgets de Estilização
- Organização de Código
- Alteração de Tema do Projeto
- Adição de Fontes Personalizadas

## Tecnologias Utilizadas

- Flutter
- Dart
